Binary Search Tree:

Beståndsdelar:
BinarySearchTree klass,
Node klass.

Node klass:
	funktioner:
	GetBalance()
	BalanceRecursion()
	GetMaxDepth()
	
	GetMaxDepth är en O(n) funktion som rapporterar största djupet som går
	att nå från denna nod (gör ett hopp per nod som utgår från denna nod,
	worst case: alla noder som utgår från Root = hela listan = n).
	
	BalanceRecursion är en O(n) funktion som anropas från GetBalance för att
	hitta balansen på den del av trädet som utgår från denna nod. Samma
	worst case som GetMaxDepth.
	
	GetBalance är den publika funktionen som använder sig av BalanceRecursion
	för att rapportera balansen. O(n) funktion eftersom den anropar
	BalanceRecursion.
